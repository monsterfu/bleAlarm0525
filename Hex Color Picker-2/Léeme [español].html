Instalación

Mover el archivo HexColorPicker.colorPicker a cualquiera de las siguientes localizaciones:

Para instalarlo sólo en su cuenta:
Copie o mueva el archivo a...
[su carpeta de usuario] → Librería → ColorPickers
(si la carpeta ColorPickers no existe, debe crearla)
Para instalarlo en su ordenador (disponible en todas las cuentas):
Copie o mueva el archivo a...
[su disco duro] → Librería → ColorPickers
(si la carpeta ColorPickers no existe, debe crearla)
Para instalarlo en algunas cuentas, pero no en todas:
Copie o mueva el archivo a...
[su disco duro] → Usuarios → [usuario] → Librería → ColorPickers
(si la carpeta ColorPickers no existe, debe crearla)
Repita este paso en cada cuenta de usuario.

Es posible que deba reiniciar las aplicaciones que tenga abiertas en ese momento para poder utilizar Hex Color Picker.

Utilización

  1. Abrir el panel de color habitual. (Puede aparecer como "Mostrar colores" (Comando + Shift + C) en los diferentes menús, o puede abrirse al hacer clic en un color para editarlo.)

  2. Haga clic en el icono de Hex Color Picker (un signo # en un círculo de color azul) en la parte superior de la ventana.

  3. Utilice el campo de texto para escribir o copiar el código hexadecimal del color actual, o utilice las herramientas estándar del panel de color (incluyendo otros medidores de color) para ajustar el color como quiera.

Abreviaturas de código hexadecimal y palabras clave de colores HTML/CSS

Puede usar una abreviatura cuando edite el código hexadecimal. Son compatibles estas formas:

  • "A" → "AAAAAA"
  • "AB" → "ABABAB"
  • "ABC" → "AABBCC"

No es necesario poner # al principio del código hexadecimal.

Si hace clic en Opciones en la parte inferior y habilita la opcióón "Reconocer los nombres de colores HTML", también puede escribir diferentes palabras clave para colores HTML/CSS. Hex Color Picker sigue las correspondencias detalladas en http://www.w3.org/TR/css3-color/#svg-color.

Consejo

Usted puede elegir si desea obtener el código hexadecimal en minúsculas o mayúsculas (si el intervalo A → F está en mayúsculas o minúsculas). Haga clic en Opciones en la parte inferior y marque o desmarque "Usar mayúsculas dígitos hexadecimales" según sea necesario.

La comprobación de actualizaciones

Dado que es probable que se le olvide visitar periódicamente el sitio web de Hex Color Picker, hay una opción que permite que Hex Color Picker se actualice automáticamente. Hex Color Picker le pedirá la primera vez que se abra si desea habilitar la actualización automática. Puede cambiar esto en cualquier momento haciendo clic en Opciones y marcando o desmarcando la casilla "Buscar actualizaciones automáticamente".

Cuando Hex Color Picker busca actualizaciones (como máximo una vez en un período de 12 horas) necesita conectar con su página web, un proceso que puede tomar uno o dos segundos, provocando un leve retraso al mostrar el panel la primera vez que se abra durante una sesión.

Cuando Hex Color Picker contacta con la página web, envía la siguiente información: la versión de Hex Color Picker, la versión de Mac OS X y el tipo de procesador (Intel o PowerPC, 32-bit o 64-bit). Estos tres datos son necesarios para dar una respuesta precisa a la pregunta "hay una actualización disponible?". No se envía información que pueda utilizarse para identificar su ordenador o su localización.

Cuestiones

Q: No puedo ver el icono en la barra de herramientas del panel de Color

R: Si no puede ver los iconos, haga clic en el botón de la barra de herramientas en la parte superior derecha para mostrar la barra de herramientas. Si puede ver el resto de los iconos, pero no el icono de Hex Color Picker, cambie el tamaño del panel de color, para que quepan todos los iconos en la barra de herramientas o haga clic en las flechas >> en la parte superior derecha para abrir un menú con iconos que sobran.

Si aún no ve el icono de Hex Color Picker, asegúrese de que está correctamente instalado, y de que ha reiniciado la aplicación después de que Hex Color Picker fue instalado, si estaba abierta en ese momento.

Q: ¿Cómo puedo utilizar Hex Color Picker en Adobe Photoshop?

R: Algunas aplicaciones usan sus propios paneles de color.

Adobe Photoshop tiene una opción para utilizar el panel color de Mac OS X: Ir al menú de Photoshop → Preferencias → General. Cambiar Selector de color de Adobe a Apple y haga clic en Aceptar.

Si utiliza otra aplicación y no puede establecer el panel de color de Mac OS X, solicite esa característica a los desarrolladores de la misma.

Q: ¿Por qué en el menú de acciones de Hex Color Picker está el comando Copiar?

R: En algunas aplicaciones, en ciertas circunstancias, los comandos del menú Edición están desactivados. El comando Copiar hace que el atajo Comando + C siga trabajando, y sólo tendrá prioridad sobre el comando Copiar habitual del menú Editar cuando el foco del teclado esté en el panel de color.
